CSV Format

# question,optionA,optionB,optionC,optionD,correct(A/B/C/D),explanation(optional) What is the capital of France?,Paris,London,Berlin,Madrid,A Photosynthesis produces which gas?,CO2,Nitrogen,Oxygen,Hydrogen,C,Plants release O2 "Find $x$ in $x^2 = 9$","$x = \pm3$","$x = 3$","$x = -3$","$x = 0$",A,"Square root of 9 is ±3"
💡 Math Tip: Use LaTeX syntax between $...$ for inline math or $$...$$ for display math. Wrap fields containing commas or math symbols in double quotes.
Columns: Question · Option A · Option B · Option C · Option D · Correct (A/B/C/D) · Explanation (optional)
You can use 2, 3, or 4 options — leave trailing ones blank. Lines starting with # are ignored.
